Beamer Animation Tutorial¶

Introduction¶

This tutorial is about how to make a technical presentations with cool animations. When I first asked around, people recommended either (1) using OpenOffice or (2) buying a Mac. This page gives a walkthrough of the another method using LaTeX, Python, and TiKZ.

And here’s an example of the presentation we’ll build in this tutorial.

Code Setup¶

To start

sudo apt-get install scons beamer tikz

The code for the tutorial lives in a github repository. You can install the library by.

$ git clone git://github.com/srush/beamer-animation.git
$ cd beamer-animation
$ sudo easy_install .

$ mkdir kmeans-slides
$ beamer-animation-quickstart

\documentclass{beamer}
\mode<presentation>
\setbeamertemplate{navigation symbols}{}
\usepackage{tikz}
% Needed for diagrams.
\def\im#1#2{
  \node(#1) [scale=#2]{\pgfbox[center,top]{\pgfuseimage{#1}}
  };}
  \input{pictures_header}
  \begin{document}
  \begin{frame}[t]{K-Means}
    \vspace{-1cm}
      \begin{figure}
          \centering
              \begin{tikzpicture}
                    \foreach \x in {1,...,10} {
                            \uncover<\x>{\im{kmeans\x}{0.6}}
                                  }
                                      \end{tikzpicture}
                                        \end{figure}
                                        \end{frame}
                                        \end{document}
